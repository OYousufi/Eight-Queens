How to play this game
	The sole purpose is to place eight queens on eight different positions such that they cannot attack each other. A queen can move in a vertical, diagonal and horizontal direction,  the number of boxes it can travel is not restricted.

How to run this game 
	keep the folder of images i.e load folder and .py file in the same folder else they wont work. Double click on .py file and that will open a console file and a game window. When you are done with the game press "ENTER" key on your keyboard and that will reset your game. I would recommend you to close the window and run game file again for better user experience.
